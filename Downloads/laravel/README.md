# News Portal Project

Бұл - Laravel және Tailwind CSS негізінде жасалған, көптілді (қазақ, орыс, ағылшын) қолдайтын толыққанды жаңалықтар порталы.

## Жобаны жаңа компьютерде іске қосу нұсқаулығы

Жобаны басқа адам алғанда (немесе жаңа компьютерге көшіргенде), келесі командаларды ретімен орындау қажет. Жобаны жұмысқа қосу үшін сіздің компьютеріңізде **PHP (8.2+)**, **Composer**, және **Node.js** орнатылған болуы шарт.

### 1-қадам. Жобаның тәуелділіктерін (dependencies) орнату
Терминалды ашып, жобаның түбірлік папкасына (`laravel` папкасы) кіріңіз де, мына командаларды орындаңыз:

```bash
composer install
npm install
```

### 2-қадам. Баптаулар файлын құру
Жобаның орта баптауларын (environment variables) орнату үшін `.env.example` файлын көшіріп, `.env` деп атаңыз:

```bash
cp .env.example .env
```

Осыдан кейін, `.env` файлын ашып, дерекқор баптауларын (DB_CONNECTION) тексеріңіз. Әдепкі бойынша SQLite қолданылады:
```env
DB_CONNECTION=sqlite
# DB_DATABASE=/absolute/path/to/database.sqlite
```

### 3-қадам. Қолданбаның кілтін генерациялау
Әрбір Laravel жобасында бірегей қауіпсіздік кілті болуы керек:

```bash
php artisan key:generate
```

### 4-қадам. Дерекқорды құру және толтыру
Мәліметтер базасын (SQLite) құрып, оны бастапқы мәліметтермен (жаңалықтар, санаттар, админ аккаунты) толтырыңыз:

```bash
# Егер database.sqlite файлы жоқ болса, оны құру керек:
touch database/database.sqlite

# Базаны іске қосу және мәліметтермен толтыру
php artisan migrate:fresh --seed
```

Бұл команда жұмыс істеп тұрған базаны тазалап, қайтадан `DatabaseSeeder` арқылы толтырады. Админ ретінде кіру мәліметтері:

- **Құпия сөз:** password

### 5-қадам. Медиа файлдарды байланыстыру (Storage link)
Жүктелген суреттер мен медиа файлдарды жалпыға қолжетімді ету үшін (public папкасына сілтеу) мына команданы орындаңыз:

```bash
php artisan storage:link
```

### 6-қадам. Жобаны іске қосу
Келесі екі команданы екі бөлек терминал терезесінде орындаңыз.

Бірінші терминалда (PHP серверін қосу):
```bash
php artisan serve
```

Екінші терминалда (CSS/JS файлдарын компиляциялау):
```bash
npm run dev
```

Осыдан кейін браузерді ашып, `http://localhost:8000` немесе `http://127.0.0.1:8000` сілтемесіне өтіп, сайтты көре аласыз!

## Қосымша пайдалы командалар

- `php artisan view:clear` — дизайн/аударма файлдарын өзгерткенде өзгерістер көрінбесе, кэшті тазалау.
- `php artisan config:clear` — `.env` файлына өзгеріс енгізгеннен кейін баптаулар кэшін тазалау.
